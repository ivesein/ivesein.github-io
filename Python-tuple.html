Python元组

Python的元组与列表相似,不同之处在于元组的元素不能修改
元组的创建很简单,只需要在小括号()中添加元素,并且用','隔开即可.
tup1 = (1, "Hello", 3.14, 'a' )
创建空元组:tup1 = ()
元组中只包含一个元素时,需要在元素后面添加逗号:
tup1 = (10,)
元组与列表,字符串类似,下标索引从0开始,可以进行截取,组合等.

访问元组的元素

元组可使用下标索引访问元组中的值,也可以进行切片截取操作,如下: 

#!/usr/bin/python
tup1 = (1, 2, 3, 4, 5, 6, 7)
print tup1[2]
print tup1[1:5]

示例结果为: 

3
(2, 3, 4, 5)

修改元组

元组中的元素值是不允许修改的,但我们可以对元组进行连接组合,如下: 

#!/usr/bin/python
#-*- coding: UTF-8 -*-

tup1 = (1, 3.14)
tup2 = ("abc", "xyz")

#以下修改元组元素的方法是非法的
#tup1[0] = 100

#创建一个新的元组
tup3 = tup1 + tup2
print tup3

示例结果为: 

(1, 3.14, "abc", "xyz")

删除元组

元组中的元素是不允许删除的,但我们可以使用del语句来删除整个元组,如下: 

#!/usr/bin/python

tup = (1, 2, 3, 4)

print tup
del tup
print tup

以上示例元组被删除后,输出变量会有异常信息,输出如下所示: 

(1, 2, 3, 4)
Traceback (most recent call last):
    file "test.py",line 7 in <module> print tup
    NameError: name 'tup' is not define

元组运算符

与字符串和列表一样,元组之间可以使用 + 和 * 进行元组的元素组合和元素重复运算,运算后产生一个新的元组:

python表达式 结果 描述
len((1, 2, 3)) 3 计算元组长度
(1, 2, 3)+(4, 5, 6) (1, 2, 3, 4, 5, 6) 连接元组
("Hi!") * 3 ("Hi!", "Hi!", "Hi!") 元组元素的重复
3 in (1, 2, 3) True 元素是否存在于元组
for x in (1, 2, 3):print x 1 2 3 迭代

元组的索引,截取

因为元组也是一个序列,所以我们可以访问元组中的指定位置的元素,也可以截取索引中的一段元素,如下:
T = (1, 2, 3)

python表达式 结果 描述
T[2] 3 访问第三个元素
T[-2] 2 反向访问,访问的倒数第二个元素
T[1:] (2, 3) 截取元素

无关闭分隔符

任意无符号的对象,只要是以逗号隔开的,默认为元组: 

#!/usr/bin/python

print 'abc', -10, 18+6.6j, "hello"
x, y = 1, 2
print "value of x, y : ", x, y

示例结果为: 

abc -10 (18+6.6j) hello
value of x, y : 1 2

元组内置函数

pyhone元组包含了以下内置函数:

函数名 描述
cmp(tuple1,tuple2) 比较两个元组元素
len(tuple) 计算元组长度(元组内元素个数)
max(tuple) 返回元组中元素最大值
min(tuple) 返回元组中元素最小值
tuple(seq) 将列表转换为元组
